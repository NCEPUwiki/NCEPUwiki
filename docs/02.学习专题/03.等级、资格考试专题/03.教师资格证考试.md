## 时间

1. 笔试报名：1月、7月上旬
2. 笔试备考：一个半月左右
3. 笔试考试3月、9月上中旬
4. 面试备考：2周左右

教资通过+普通话通过=教资认定

报名条件：大三及以上。报名材料审核成功后才能缴费

## 笔试

备考参考书：粉笔、中公

备考顺序：教育教学知识-综合素质-专业课

方法：听课+APP刷题，考前1-2周背大题背作文

注：
1. 背诵可以是关键词+一两句话
2. 网上资料大题预测（姜姜老师、卢姨）
3. 科二推荐中公晓飞老师

## 面试

备考资料：电子版文字资料+讲课视频

结构化面试：背资料+理解（建议：大声背诵）

备课教案及试讲：
1. 系统学习题型
2. 根据具体题目，参考标准教案和讲课视频
3. 自己根据题目写教案，一定要站在讲台上试讲（抖音：教师考试王老师）

答辩：问什么答什么，一般根据试讲内容提问

### 面试技巧

心态上：放弃上台包袱
1. 我要好好表现，让他们觉很我很牛。
2. 磕巴说错了，被笑话怎么办？
3. 大家都不看PPT，好尴尬。

降低心理预期:
1. 奔着“我就是丢脸来了”的绝妙心态
2. 只要站上讲台，把需要讲述的内容讲完就成功
3. 接受不完美，一点小失误不影响整体

循序渐进地进步：
1. 这次上台发言声音不颤抖了
2. 可以脱稿与同学们眼神交流
3. 自然地加上手势动作，与观众进行互动

行动上：强准备

常见问题：上台前紧张、焦虑，准备不充分对所讲内容不熟悉

办法：写好+演好逐字稿
1. 写逐字稿时，一定要有逻辑线
2. 去掉书面化的文字（碎嘴部分也写）
3. 对着镜子练习逐字稿
4. 模拟现场，大声讲出内容（空教室、找朋友听）

tips: 
1. 多去参加一些校级的答辩或比赛现场
2. 主动参加各种活动，大胆展示自己
3. 人生没有那么多观众，多关注内容
4. 锻炼身体，穿着得体
