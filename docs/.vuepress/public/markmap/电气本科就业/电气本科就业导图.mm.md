# 电气本科就业

## 主要就业方向

### 电网（占60%）

#### 国网

##### 岗位信息

- 调度岗：需要倒班，凌晨也必须在岗，工作强度大，值班时几乎没有睡觉的可能；不涉及外出，在核心办公大厅工作。对学历要求很高，但薪资水平更高，升职速度也更快。
- 运维岗（变电站值班）：需要值班，但强度远低于调度岗，值班期间可以休息睡觉。
- 其他常规岗位：正常情况下执行国家工作日作息，基本不会凌晨上班；仅在发生重大事故时，才需要参与抢修加班。

##### 备考报班

- 建议从大三开始系统准备。机构报名后可以一直学习，直到考试。
