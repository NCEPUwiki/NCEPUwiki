## Linux

### 安装

一般情况下，发行版预装了Python的某个版本（如果只有命令行可能不会预装，因为属于某些组件的依赖），可以试试
```python3 --version```查看。

如果发现命令未找到，可以试试直接使用发行版的包管理器查找。此步请参阅发行版wiki或手册。

### pip/虚拟环境

在特别早期的版本中是可以直接使用```pip```来安装包的，后期的时候会提示不允许这么干，据笔者刚刚试验，甚至已经无法使用了。

```
$ python3 -m pip
/usr/bin/python3: No module named pip
$ pip
bash: pip: 未找到命令
```

这时需要给所需目录弄一个虚拟环境，首先运行```python3 -m venv /path/to/destination```（将后面的目录换成所需目录），
然后进入目录，运行```source bin/activate```进入虚拟环境。这时就可以直接使用```pip```等命令了，不过包的缓存会在本目录
下，请不要删除。
